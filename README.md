# Sistema de Gerenciamento de Notificações

Este é um sistema de gerenciamento de notificações que permite aos usuários personalizar suas preferências de notificação em diferentes canais.

## Visão Geral

O sistema foi construído com as seguintes tecnologias:

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

## Funcionalidades

- Visualizar e alternar preferências de notificação.
- Visualizar estatísticas de canais de notificação ativos.
- Layout responsivo para desktop e dispositivos móveis.

## Começando

Para executar o projeto localmente, siga estas etapas:

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. **Instale as dependências:**

   ```bash
   npm install
   # ou
   yarn install
   # ou
   pnpm install
   ```

3. **Execute o servidor de desenvolvimento:**

   ```bash
   npm run dev
   # ou
   yarn dev
   # ou
   pnpm dev
   ```

Abra [http://localhost:3000](http://localhost:3000) em seu navegador para ver o resultado.
